## Overview

Create a documentation site for the Furtherfield LARP playbook using Fumadocs, styled to match the marketing site's design language. The playbook content will be presented in a software documentation format for easy navigation and reading.

### Key Capabilities

- **Set up Fumadocs documentation framework in the monorepo**
- **Customize Fumadocs theme to match marketing site design (dark theme, acid accents, Geist Mono typography)**
- **Import and structure playbook content for documentation-style presentation**
- **Ensure consistent design language with treatment colors and callout styles**
